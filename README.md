# Job Market Analysis in NYC

This project analyzes job market trends in New York City using data analysis techniques to uncover insights about job demand, salaries, and required skills.

## Project Overview

Understanding job market trends helps job seekers and companies make better decisions. This project explores job listings data to identify patterns in job demand, salary distribution, and required skills.

## Objectives

- Analyze job distribution by role and industry
- Identify high-demand skills
- Explore salary trends
- Provide insights into job market behavior

## Tools and Technologies

- Python
- Pandas
- Matplotlib
- Seaborn

## Workflow

1. Data loading and cleaning  
2. Exploratory data analysis  
3. Visualization of job trends  
4. Insight generation  

## Key Insights

- Certain job roles dominate the market
- High-paying jobs often require specialized technical skills
- Skill demand varies across industries
- Salary distribution shows significant variation
